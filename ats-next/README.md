# ats-next

Next.js 14 frontend + Prisma/SQLite backend for the ATS application tracker.

See the [root README](../README.md) for project overview, stack, Docker instructions, and data model.

## Quick start

```bash
npm install
npx prisma generate
npm run dev
```

Open http://localhost:3000.

## Scripts

- `npm run dev` — dev server
- `npm run build` — production build
- `npm start` — run production build
- `npm run lint` — ESLint
- `npm test` — Jest
